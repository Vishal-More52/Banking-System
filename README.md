Banking System Application (Java)
A robust, console-based Banking System built with Pure Java. This project demonstrates the practical application of Core Java concepts, focusing on professional software architecture using the Repository Pattern and Service Layer separation.

🚀 Key Features
Account Management: Functionality to open and manage customer accounts.

Transaction Suite:

Deposit: Securely add funds to an account.

Withdrawal: Remove funds with real-time balance validation.

Fund Transfer: Transfer money between two internal accounts.

Statement & Search:

Generate a full transaction history (Statement).

Search and filter accounts by customer name.

Data Integrity: Built-in input validation for account numbers and transaction amounts.

Exception Handling: Custom implementation for handling insufficient funds and invalid operations.

🛠️ Technical Concepts Applied
Object-Oriented Programming: Deep dive into Encapsulation and Abstraction.

Java Records: Used for modern, immutable data representation (Account and Transaction models).

Collections Framework: Efficient data management using ArrayList and HashMap.

Repository Pattern: Decoupling data storage logic from business logic.

Service Layer: Centralized business logic for banking operations.

Java Time API: Using LocalDateTime for accurate transaction logging.
